🚀 Crewmate Sprite Previews — Among Us Style

Red crewmate · 40×52px canvas · displayed at 3× scale (120×156px)

How to export as PNG for GameMaker:
1. Open this file in a browser (just double-click sprite_preview.html)
2. Right-click a sprite → Save Image As → saves as .svg
3. Open the SVG in Inkscape (free) → File → Export PNG Image → set width 40 → Export
   Or use svgexport.io online — upload, set 40×52, download PNG.
4. In GameMaker: right-click Sprites → Create Sprite → Import the PNG → rename to match the filename below.
